## 说明
本repo为biendata代码生成竞赛的解题代码。(竞赛链接)[https://www.biendata.xyz/competition/siemens-ai/]

## 运行方式

本代码仓库所有代码均在python3.9.17环境下编写、调试。
```bash
pyhon=3.9.17
```


安装依赖：

```bash
pip install -r requirements.txt
```

运行初赛赛题生成：

```bash
python ./pre_round_generation.py
```

运行生成服务：

```bash
python main.py
```
